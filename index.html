Конфигурация Vim

Установка Vim

sudo apt install vim

curl -fLo ~/.vim/autoload/plug.vim --create-dirs \ https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

Базовая конфигурация

В начале создайте конфиг:

touch ~/.vimrc

Отлично! Теперь начните его редактировать через сам Vim:

vim ~/.vimrc

Изначально вам необходимо прописать отображение синтаксиса:

syntax on

И поставьте кодировку:

set encoding=utf8

Ниже есть ещё интересные настройки:

set mouse=a " Мышка
set number " Показ чисел
set relativenumber " В обратной последовательсти
set tabstop=2
set shiftwidth=2 " Количество пробелов в табе
set clipboard=unnamed " Системный буфер обмена
set cursorline " Горизонтальное выделение текста, также есть вертикальное (set cursorcolumn)
set nocompatible " Отключить поддержку с Vi

Установка плагинов

Так же можете установить различные плагины с помощью Vim-Plug:

call plug#begin()
Plug 'preservim/nerdtree'
call plug#end()
